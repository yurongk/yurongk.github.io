- nvm version 查看 nvm 版本
- nvm ls 查看 node 版本
- nvm install <version> 下载对应 node 版本
- nvm use <version> 切换到对应 node 版本
- node -v 确认查看一下
  - 目前装了 14.0.0 18.16.0(官网的稳定版本)
  - 如果用 nvm use 命令切换 node 版本时遇到 exit status 5，请以管理员模式重启 cmd 再运行。
