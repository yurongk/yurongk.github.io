1. npm i patch-package --save-dev

2. package.json
   ```json
    "scripts": {
      "postinstall": "patch-package"
    }
    ```

3. 修改node_modules里面的代码

4. 执行命令：npx patch-package <package_name>

验证：
新增 patches 目录下有 

```
<package_name>+<version>.patch

npx npkill -> npm install 

node_modules 内容自动被修改了
```
